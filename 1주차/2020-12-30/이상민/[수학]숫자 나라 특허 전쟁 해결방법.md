## [입력]
- 첫째 줄에 자연수 N을 입력합니다.
```math
(1 <= N <= 100,000)
```
## [출력]
- N보다 작은 자연수 중에서 특허가 걸려 마음대로 쓸 수 없는 수들의 합을 출력합니다.
## [입력 예시]
```
10
```
## [출력 예시]
```
23
```

## [풀이]

### n을 입력받고, 1부터 n까지 3과 5의 배수를 answer에 저장한다.
``` python
N = int(input())

answer = [i for i in range(N) if i % 3 == 0 or i % 5 == 0]

print(sum(answer))
```
