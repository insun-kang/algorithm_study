## [입력]
- 고객이 원하는 단백질 바 한 개의 단백질 함량을 나타내는 정수 N을 입력합니다.
``` math
(1 <= N ≤ 10,000)
```
## [출력]
- 한 개의 단백질 바에 들어가는 재료 큐브의 개수를 한 줄에 공백으로 구분하여 출력합니다. 출력의 순서는 땅콩 큐브, 닭가슴살 큐브, 순수 단백질 큐브입니다.
- 만약, 만들 수 없는 경우 -1을 출력합니다.
## [입력 예시]
```
300
```

## [출력 예시]
```
1 1 1
```

## [풀이]

### 1. 원하는 단백질 함유량을 입력
### 2. 각 큐브의 단백질 함유량을 cubes에 저장하고

protein = int(input())
